"Bounty Hunter".

The purpose of the application: to raise the level of the fight against corruption with the help of citizens.

The essence of the application: Citizens anonymously leave a statement about the fact of corruption (bribery, embezzlement of public funds, etc.). This application contains fields about / for:

  • institution / company in which the fact of corruption occurred,
  • the person or persons involved in this act of corruption,
  • a description of this corruption action,
  • files proving this corruption action (video, audio, photo, etc.).

For a user to leave a request, he needs to log in to the system. All his personal data will be hidden from other users. In the system, the user will act under a pseudonym. For authorization, the user must fill in the fields:

  • Name
  • Surname
  • Сontact phone number
  • e-mail
  • IIN

After sending the application, it is checked. In case of accumulation of a certain number of applications, a check is sent to this organization / company.

This project will contain pages:

1) The main page in which news, information about the situation in the Country, city, region, etc. will be displayed. This page will be able to display information on the selected region.

2) Registration / authorization page.

  • Name
  • Surname
  • Password
  • Pseudonym (If this field is not filled in, the system will issue itself)
  • Сontact phone number
  • e-mail
  • IIN

The authorization page will have fields:

  • e-mai
  • Password

3) Page for filling out a statement of the fact of corruption.

This page will have fields

  • institution / company in which the fact of corruption occurred
  • the person or persons involved in this act of corruption
  • a description of this corruption action
  • files proving this corruption action (video, audio, photo, etc.)
